# 学生画像

- 考生类型：北京考生，希望走法学或相近规则治理方向
- 分数位置：560 分属于要讲策略、不能只看学校名字的区间
- 城市偏好：优先北京
- 家庭期待：看重体制、法考、公务员、法务这类相对稳定的路径
- 个人诉求：想学法学，但又担心分数不上不下、最后卡在学校和专业之间

# 核心诉求

- 想把法学作为主方向，而不是随便找个不限选科专业替代
- 希望兼顾北京城市资源和后续考试路径
- 最担心录到学校了，但实际走不到法学那条路

# 风险判断

- 法学不是“录到学校就结束”的专业，后面法考、考公、考研都会继续筛人
- 北京法学方向如果只看学校名气，不看专业组和培养方向，判断很容易失真
- 560 分这个位置，学校优先和专业优先往往不能两头都要
- 如果本人并不愿意持续考试，法学这条路后期容易产生心理落差

# 推荐方向

- 先看目标学校法学所在专业组，再看学校平台，不要反过来
- 如果目标是体制路线，可以接受“学校平台好但法学不绝对顶尖”的方案
- 如果目标是法律实务或后续深造，专业纯度和学校法学平台都要一起看
- 如果法学不够稳，可以同步比较政治学、行政管理、社会治理、中文类等相近考试路线专业

# 不推荐方向

- 不建议把“学法学”简单理解成“以后自然进体制”
- 不建议为了留北京，接受与法律路径差得太远的不限选科专业
- 不建议在不了解自己是否能长期卷考试的前提下盲目把法学放第一顺位

# 冲稳保思路

- 冲：北京平台更强、法学或相近专业组边缘可搏的目标
- 稳：北京本地能承接法学、治理、行政、中文等考试路线的学校和专业组
- 保：北京公办本科里自己能接受的规则型或公共事务型方向
- 如果法学纯度太勉强，就要加一层“平台保住、方向相近”的现实备选

# 情绪安抚与沟通重点

- 560 分不是法学没戏，而是不能再靠幻想做决定
- 真正怕的不是学不到法学，而是你以为自己学到了，结果后面根本不愿意继续卷
- 这类志愿题不是一句“能不能上”就能解决的，要看你想走到哪一步

# 张雪峰风格解释

想学法学，先别急着问学校漂不漂亮，你先问自己一句：后面几年你到底愿不愿意继续卷。法学不是那种“录上就结束”的专业，很多人的苦不是在填志愿的时候，是在毕业以后才开始。560 分在北京不是完全不能抓法学，但你得想清楚，是要一个更像法律赛道的专业，还是要一个更像北京平台的学校，这两个目标很多时候没法同时吃满。

# 下一步最值得补充的信息

- 目标学校法学所在专业组
- 当年北京对应位次
- 是否接受考研和法考路线
- 是否接受法学的相近替代专业
