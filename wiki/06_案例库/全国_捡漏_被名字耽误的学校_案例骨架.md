# 学生画像

- 分数：接近行业特色强校的报考边缘
- 诉求：想要平台更好的学校，但不执着于“名字好听”
- 特征：愿意接受行业特色明显、专业名称不时髦的学校

# 典型识别思路

- 学校名字容易让家长误以为“传统行业、未来不行”
- 但学校平台、行业认可度、特色专业实力并不弱
- 用户如果只看名字，很容易错过真实机会

# 可作为候选观察对象的院校类型

- 地矿油类行业特色高校
- 带“地质 / 矿业 / 石油 / 农林 / 海洋”等名称、但平台并不低的学校
- 区域行业型强校、地方特色强校、名字不潮但出口不差的院校

# AI 输出要求

- 不要把“名字被低估”写成录取确定性
- 要明确说明：这类学校是否构成捡漏，仍需结合本省位次、专业组和用户是否接受行业属性
- 推荐时必须同时讲清楚：平台优势是什么、行业约束是什么

# 风险提醒

- 如果用户根本不接受相关行业方向，名字被低估也不等于适合他
- 有的平台是强，但专业约束和职业方向也更明确
- 不能把“名字不热门”误解成“人人都能捡到”

# 张雪峰风格解释

有些学校最吃亏的地方，不是实力不行，是名字把自己耽误了。家长一看“地质”“矿业”“石油”，脑子里先冒出来的不是平台，而是“土”“辛苦”“不热门”。但现实是，很多这种学校平台并不低，行业认可度也不弱。问题不在学校行不行，问题在你能不能接受它背后的行业属性。能接受，它就是机会；接受不了，它再被低估也跟你没关系。
