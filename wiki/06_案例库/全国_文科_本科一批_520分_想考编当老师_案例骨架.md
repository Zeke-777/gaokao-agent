# 学生画像

- 分数段：505–535 分
- 选科：历史方向
- 家庭状态：希望回省内或回地市、县城，求稳定
- 本人诉求：可以接受做老师，但不希望多年折腾仍然上不了编

# 核心诉求

- 目标非常明确：想考编，当老师
- 最关心：到底选学校更重要，还是选学科方向更重要
- 希望毕业后能尽量顺着本地路径走，不想跑太远

# 风险判断

- 师范不等于自动上编，最后还是要考试
- 同样是师范，学科方向不同，编制难度差很多
- 如果只图学校名字，不看学科缺口和本地需求，后面会很被动

# 推荐方向

- 先看本地需求，再看学科方向，再看学校
- 数学、英语、体育、音乐、心理等方向，有些地方的缺口会更明显
- 如果一心想回本地，本省认可度和地方口碑往往比全国名气更重要

# 张雪峰风格解释

想考编当老师，这是很务实的路子，但别把“师范”两个字想得太简单。学校当然重要，可很多时候真正决定你上不上编的，是你教什么。语文历史这种热门方向，一个岗位可能一堆人抢；数学、英语、体育这些地方缺口更大，反而更容易落地。所以这道题别只问学校，先问岗位。
