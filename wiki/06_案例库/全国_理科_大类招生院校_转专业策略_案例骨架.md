# 学生画像

- 分数段：565–595 分
- 目标：进更好的学校，再尝试在校转热门专业
- 核心想法：先上平台，再靠大学成绩抢专业
- 担忧：不知道这种策略到底是机会还是坑

# 核心诉求

- 想“低门槛进好校”
- 想利用大类招生或转专业规则换到更热门的方向
- 不想高考时分不够，大学里也再输一次

# 风险判断

- 大类招生不等于自动进入热门专业
- 热门方向分流和转专业通常非常卷，很多学校还会有成绩和名额限制
- 如果你只是靠高考边缘进校，大学里未必还能保持同样竞争力

# 推荐方向

- 只有在明确能接受在校继续卷、并且学习能力确实不错时，才把这当主策略
- 如果本人本来就不算特别能卷，直接选相对确定的专业通常更稳
- 真要走这条路，必须提前查清楚分流、转专业和大类内专业构成

# 张雪峰风格解释

大类招生这件事，很多人把它当捷径，实际上它更像二次选拔。你高考已经卷过一次了，进学校以后还得再卷一次，而且周围都是跟你一样甚至比你更能卷的人。要是你真有把握大学里还能稳住前排，这条路可以搏；要是只是觉得“先进学校再说”，那很可能最后连原来想要的方向都没摸到。
