### 4.4 系统优势

1. **零宕机** — 110 题连续测试，无任何崩溃或错误
2. **智能路由** — LLM 正确区分了"需要实时数据"vs"知识库可回答"的题目
3. **多源验证** — 复杂查询自动多轮搜索，覆盖多个数据平台
4. **自适应深度** — 批次线 1-2 调用/12s，志愿策略可达 17 调用/79s
5. **知识库互补** — 政策规则和就业前景类题目充分利用 wiki 内容

### 4.5 改进空间

| 发现                 | 影响  | 建议                   |
| ------------------ | --- | -------------------- |
| 复杂查询耗时偏长（最长 82s）   | 中   | 限制最大搜索轮次到 6          |
| 排名对比类搜索次数偏多（均 6.5） | 低   | 提示词引导批量搜索            |
| 志愿策略类调用数差异大（3-17）  | 低   | 标准化冲稳保查询模板           |
| 知识库对录取分数查询无贡献      | 低   | 正常 — 录取分走 search_web |
