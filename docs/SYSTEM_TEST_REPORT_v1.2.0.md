## 四、抽样实测结果

从 120 道题中抽取 4 道覆盖不同类别的题目进行完整测试：

### 实测 1：清华大学 2025 山东录取分

| 维度 | 详情 |
|------|------|
| **类别** | 院校录取查询 |
| **问题** | 清华大学 2025 年在山东的录取分数线是多少？物理类最低位次？ |
| **工具调用** | search_web × 2 |
| **数据来源** | 6617.com、新浪财经（引山东省教育招生考试院官方投档表） |
| **耗时** | 27.7s |
| **核心数据** | 685 分 / 全省 140 名（理科试验班） |
| **数据验证** | 多源交叉验证 ✅ |
| **投档线/专业线区分** | 明确区分 ✅（"最低消费 685，招牌菜得加价"） |
| **趋势分析** | 2023-2025 三年对比表 ✅ |
| **建议质量** | 强基计划/提前批备选方案 + 位次差分析 ✅ |
| **免责声明** | 提醒以省考试院官方为准 ✅ |
| **张雪峰风格** | "名校的冷板凳和强校的热门专业，后者的投入产出比更高" ✅ |

**答案摘要**：
> 清华大学 2025 年在山东最低 685 分，位次 140 名（理科试验班）。不同专业分差大：文科试验班需全省前 50 名，临床医学位次更靠前。近三年位次稳定在前 150 名以内。位次差几十名的，建议考虑复旦、上交、浙大、中科大的王牌专业。

### 实测 3：计算机 vs 人工智能就业前景

| 维度 | 详情 |
|------|------|
| **类别** | 就业前景 |
| **问题** | 计算机专业现在还值得学吗？2026 年就业前景？和人工智能比哪个更好？普通家庭投入产出比？ |
| **工具调用** | search_knowledge × 2 + search_web × 2 |
| **数据来源** | wiki 人工智能专业库、6617.com 对比文章、Data Application Lab |
| **耗时** | 37.2s |
| **核心发现** | AI 专业更窄更深（高科技公司为主），计算机就业面更广。AI 需要强数学基础（高考数学 120+） |
| **知识库贡献** | wiki 提供了 AI 专业详细画像：课程、适合/不适合人群、不适合数学弱的学生 |
| **免责声明** | 无（知识型回答不需要） |

**答案摘要**：
> 计算机仍然值得学，但别指望"随便学学就能高薪"。AI 专业更适合数学强+愿意科研的学生；计算机就业面更宽（互联网/金融/制造/政府都能去）。普通家庭建议报计算机——4 年本科即可就业，AI 通常需要读到研究生。投入产出比：计算机本科起薪 8-15K，5 年经验 20-40K。

## 七、建议改进项
